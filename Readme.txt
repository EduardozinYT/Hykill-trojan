Hykill: Malware Educativo Destrutivo

AVISO! Este código é um malware educativo extremamente destrutivo. Sua execução pode danificar gravemente o sistema operacional e apagar seus dados.

Este malware não deve ser executado em um computador real. Use-o apenas em um ambiente virtual controlado, como o VirtualBox, para fins de aprendizado e pesquisa em segurança cibernética.

O que o Hykill faz:

    Este malware é capaz de:
        Excluir arquivos e pastas importantes do sistema.
        Corromper o sistema operacional, tornando o computador inoperante.
        Formatar o disco rígido, apagando todos os dados. OBS esse aviso não é tão certeiro mais vou colocar

Como usar o Hykill com segurança:

    Leia este README com atenção antes de executar o código.
    Faça backup de todos os seus dados importantes antes de continuar.
    Execute o código em um ambiente virtual isolado.
